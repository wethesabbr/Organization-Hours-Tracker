Organization Hours Tracker

This project is an Excel-based hours tracker that records and calculates how long each organization member stays in the office each day. It helps track productivity and attendance.

Features
- Track daily time in / time out
- Automatically calculate total hours rendered
- Supports multiple time entries per day
- Easy-to-use Excel interface

How It Works
1. Enter the Time In and Time Out of each member.
2. The sheet calculates the total number of hours worked.
3. Data can be summarized weekly or monthly for reports.

Requirements
- Excel

How to Use
1. Download or clone this repository.
2. Open the `.xlsx` file in Excel.
3. Start logging member hours.

Future Improvements
- Add charts to visualize attendance trends
- Automate data export to CSV
